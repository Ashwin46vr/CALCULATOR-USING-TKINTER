Assignment 6 – Tkinter Calculator

This project is a straightforward calculator created through the use of Python Tkinter.
It is capable of performing the most common mathematical operations which are:

Addition

Subtraction

Multiplication

Division

Files Included

calculator.py

README.md

How to Run the Program

Confirm that you have installed Python.

Execute the file with:

python calculator.py

Features

Numeric buttons for the digits 0–9

Sign buttons for +, -, *, /

The "default" button displays the result

"CLEAR" button empties the entire input

Invalid inputs lead to "Error" notifications

Submission Steps

Place calculator.py and README.md into one folder.

Right-click the folder → Send to → Compressed (zip).

Transfer the ZIP file to Google Drive.

Enable the option "Anyone with the link can view".

Provide the drive link as submission.
